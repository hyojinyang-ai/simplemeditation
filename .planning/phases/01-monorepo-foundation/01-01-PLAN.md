<task type="auto">
  <name>Task 2: Add Turborepo and configure tasks</name>
  <files>package.json, turbo.json</files>
  <action>
    Update root package.json to add turbo as devDependency (^2.8.0), set packageManager field to "pnpm@10.28.0" (required by Turborepo 2.0+), update name to "simple-meditation-monorepo", and update scripts to use turbo commands:
    - "dev": "turbo dev"
    - "build": "turbo build"
    - "lint": "turbo lint"
    - "test": "turbo test"

    Create turbo.json with task definitions following research examples:
    - build: dependsOn ^build, outputs ["dist/**", ".next/**", "!.next/cache/**"]
    - dev: cache false, persistent true
    - lint: dependsOn ^build
    - test: dependsOn build, outputs ["coverage/**"]

    Include $schema field pointing to https://turbo.build/schema.json for IDE validation.

    CRITICAL: Without outputs defined, Turborepo will NOT cache task results. Each task that produces files must explicitly list output patterns.
  </action>

<task type="auto">
  <name>Task 3: Create typescript-config shared package</name>
  <files>packages/typescript-config/package.json, packages/typescript-config/base.json, packages/typescript-config/react.json</files>
  <action>
    Create packages/typescript-config/ directory with three files:

    1. package.json:
    ```json
    {
      "name": "@repo/typescript-config",
      "version": "0.0.0",
      "private": true,
      "exports": {
        "./base.json": "./base.json",
        "./react.json": "./react.json"
      }
    }
    ```

    2. base.json (shared TypeScript config for all packages):
    ```json
    {
      "compilerOptions": {
        "target": "ES2020",
        "module": "ESNext",
        "lib": ["ES2020"],
        "moduleResolution": "bundler",
        "strict": true,
        "skipLibCheck": true,
        "esModuleInterop": true,
        "resolveJsonModule": true,
        "isolatedModules": true,
        "declaration": true,
        "declarationMap": true,
        "incremental": true
      }
    }
    ```

    3. react.json (extends base with React-specific options):
    ```json
    {
      "extends": "./base.json",
      "compilerOptions": {
        "jsx": "react-jsx",
        "lib": ["ES2020", "DOM", "DOM.Iterable"]
      }
    }
    ```

    These configs will be imported by all workspace packages using `"extends": "@repo/typescript-config/base.json"` pattern per Turborepo recommendations.
  </action>
  <verify>
    <automated>test -f packages/typescript-config/package.json && test -f packages/typescript-config/base.json && test -f packages/typescript-config/react.json && grep -q '"@repo/typescript-config"' packages/typescript-config/package.json</automated>
  </verify>
  <done>typescript-config package exists with base.json and react.json configurations, exports field properly configured</done>
</task>
