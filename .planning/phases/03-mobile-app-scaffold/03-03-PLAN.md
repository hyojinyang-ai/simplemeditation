Create lib/storage.ts implementing StateStorage interface:
```typescript
import { MMKV } from 'react-native-mmkv';
import type { StateStorage } from '@repo/meditation-core';

// Create MMKV instance (singleton)
const mmkv = new MMKV();

/**
 * MMKV storage adapter for Zustand persist middleware.
 * Implements StateStorage interface from @repo/meditation-core.
 */
export const mmkvStorage: StateStorage = {
  getItem: (name: string): string | null => {
    const value = mmkv.getString(name);
    return value ?? null;
  },
  setItem: (name: string, value: string): void => {
    mmkv.set(name, value);
  },
  removeItem: (name: string): void => {
    mmkv.delete(name);
  },
};
```

Follow Pattern 2 from RESEARCH.md exactly. MMKV.getString returns `string | undefined`, must convert undefined to null for StateStorage interface compatibility.

react-native-mmkv is a native module, so after install:
1. Stop metro bundler
2. Run `npx expo prebuild --clean` to regenerate native projects
3. Rebuild app with `npx expo run:ios`

This generates ios/ and android/ folders but keeps managed workflow (CNG pattern).

  </files>
  <action>
Create lib/store.ts instantiating meditation store with MMKV:
```typescript
import { create } from 'zustand';
import { createMeditationStore } from '@repo/meditation-core';
import { mmkvStorage } from './storage';

/**
 * Mobile app meditation store with MMKV persistence.
 * Uses createMeditationStore factory from @repo/meditation-core.
 */
export const useMeditationStore = create(
  createMeditationStore(mmkvStorage, 'mobile-meditation-entries')
);
```

Use storage key 'mobile-meditation-entries' (different from web's 'zen-mood-entries-v2') to avoid conflicts if testing locally with same Supabase account later.

This pattern matches web app's apps/web/src/lib/meditation-store.ts but with MMKV instead of localStorage.

Zustand's persist middleware:
- Automatically serializes state to JSON
- Calls mmkvStorage.setItem on state changes
- Calls mmkvStorage.getItem on store initialization (hydration)
- Handles rehydration without manual intervention
