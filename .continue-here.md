## Key Decisions Made

1. **Analytics Approach:** Used Vercel Analytics with custom events instead of GA4 for simplicity and privacy compliance
2. **UI Evolution:** Started with glassmorphism for premium feel, then simplified to clean minimal design based on user preference
3. **Auto-play:** Removed manual play button to streamline meditation flow
4. **Session Protection:** Added confirmation only for "Home" navigation during meditation (other tabs don't interrupt)
5. **State Management:** Used existing Zustand store for `isMeditating` state rather than React Context

## Technical Context

### Analytics Events Tracked:
- `page_view` - All route navigation
- `session_start` - Meditation begins
- `session_complete` - Full session finished with mood data
- `session_abandoned` - User exits early (with completion %)
- `pre_mood_selected` / `post_mood_selected` - Mood tracking with transformations
- `sound_changed` - Sound preference changes
- `quote_saved` - Quote engagement
- `pull_to_refresh` - Mobile gesture usage

### Flow Changes:
**Old:** Mood → Duration → Sound → **Play Button** → Meditation → Reflect → Quote
**New:** Mood → Duration → Sound → **Auto-start** Meditation → Reflect → Quote

### Session Protection Logic:
```typescript
// In BottomNav.tsx
if (to === '/' && isMeditating && pathname !== '/') {
  e.preventDefault();
  setShowConfirm(true); // Show confirmation dialog
}
```
